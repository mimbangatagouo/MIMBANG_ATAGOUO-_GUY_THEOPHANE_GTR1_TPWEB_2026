Introduction

Ce mini-site a ete creer dans le cadre du cours de TP programmation WEB

Il permet de decouvrire la creation de plusieurs pages HTML relieess entre elles

Objectif du mini-site

L'objectif est d'apprendre a structurer un petit projet WEB avec plusieur fichiers

Pages disponibles

  • La page d'acceuil presente le contexe generale du mini-site
  • La page presentation contien les informations personnelles de l'etudiant
  • La page Contact contien les informations de contact de l'etudiant

J'apprend la programmation WEB parce qu'elle me permettra de creer des interfaces utiles pour les services reseau et les applications d'entreprise
